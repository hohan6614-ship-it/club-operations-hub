탭 1 · 시간표 조율

시간표를 만들고 참여자를 넣은 뒤, 사람별 불가시간을 입력해서 30분 단위로 가능한 시간을 계산합니다. 일정 확정 시 시간표 이름이 기본 일정명으로 들어가고, 확정된 일정은 다른 시간표에도 전역 불가로 반영됩니다.

시간표 생성 드래그 입력 전역 확정 일정 반영 연습실 배정

1) 사람 관리

전역 사람 목록입니다. 한 번 추가한 사람은 여러 시간표에서 다시 선택할 수 있습니다.

2) 시간표 생성 / 선택

어떤 목적으로 일정을 맞추는지 시간표를 만들고 참여자를 연결하세요.

3) 참여자 / 전역 확정 일정

선택한 시간표의 참여자를 관리하고, 모든 시간표에 반영되는 확정 일정을 확인합니다.

전역 확정 일정

4) 입력 / 보기 설정

불가시간 입력 모드와 결과 표시 기간을 조절합니다. 이동시간은 자동 계산하지 않고 직접 입력합니다.

드래그 시작 칸이 비어 있으면 추가, 이미 막혀 있으면 해제됩니다. 같은 사람에 대해 율전/명륜 일정을 번갈아 입력할 수 있습니다.

5) 가능 인원 결과

선택한 시간표 참여자 기준으로 30분 단위 가능 인원을 계산합니다. 확정 일정은 다른 시간표에서 자동 반영됩니다.

0명 적음 중간 많음 전원 가능

6) 일정 확정 / 연습실

후보 시간 중 하나를 확정하면 이 시간표 이름이 기본 일정명으로 들어가고, 참여자 전체의 전역 일정으로 반영됩니다.
